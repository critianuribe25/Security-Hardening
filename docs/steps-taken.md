# Steps Taken — Windows Security Hardening

1. Opened **Windows Security** from the Start menu.
2. Reviewed the Home dashboard to check overall protection status.
3. Navigated to **Virus & threat protection**:
   - Verified real-time protection was enabled.
   - Ran a quick scan.
4. Opened **Firewall & network protection**:
   - Confirmed firewall was enabled for Domain, Private, and Public networks.
5. Opened **App & browser control**:
   - Verified SmartScreen and reputation-based protection were turned on.
6. Searched for **User Account Control (UAC)** settings:
   - Ensured UAC was set to the recommended level.
7. Opened **Windows Update** in Settings:
   - Checked for and installed any pending updates.
8. Reviewed **Accounts → Family & other users**:
   - Verified only appropriate admin and standard accounts existed.
